## 1. 系统概述

IT 资产管理系统是一套面向企业 IT 资产全生命周期管理的 Web 应用，覆盖资产登记、入库出库、领用审批、维修保养、报废处置、折旧计算、盘点核对等完整业务链路。

**技术栈**: Vue 3 + Element Plus（前端）/ FastAPI + SQLite（后端）  
**访问地址**: `http://localhost:5173`  
**默认账号**: `admin` / `admin123`

## 3. 角色与权限

系统内置 5 种角色，不同角色看到的菜单和可执行的操作不同：

| 角色 | 标识 | 权限范围 |
|------|------|----------|
| 超级管理员 | `super_admin` | 全部功能，可管理系统配置和用户 |
| 资产管理员 | `asset_admin` | 资产管理、出入库、审批、报表，不可管理系统配置 |
| 部门主管 | `dept_manager` | 查看本部门资产，提交/审批领用申请 |
| 普通用户 | `user` | 查看资产，提交领用申请 |
| 审计员 | `auditor` | 只读查看所有模块 |

### 3.1 权限操作说明

- **超级管理员** 和 **资产管理员** 可执行所有资产的增删改查、出入库确认、报废操作
- **部门主管** 只能审批本部门的领用申请
- **普通用户** 只能提交领用申请，不能审批
- **审计员** 仅可查看，不能执行任何写入操作

## 5. 资产管理

### 5.1 资产列表

**路径**: 资产管理 → 资产列表

**功能**:
- **搜索**: 按资产编号、名称、分类、部门、状态筛选
- **查看**: 点击行查看资产详情
- **编辑**: 点击「编辑」修改资产信息
- **删除**: 点击「删除」移除资产（需确认）
- **时间线**: 点击「时间线」查看资产全生命周期
- **导出**: 点击「导出 Excel」导出当前筛选结果为 CSV 文件

### 5.2 资产登记

**路径**: 资产管理 → 资产登记

**操作步骤**:
1. 填写资产基本信息（编号、名称、分类、品牌、型号等）
2. 选择分类时，只能选择**二级分类**（即已有子分类的分类不可选）
3. 填写采购信息（采购日期、价格、使用年限）
4. 选择折旧方法：
   - **直线法**: 按使用年限平均分摊折旧
   - **一次性**: 一次性全额折旧
5. 点击「保存」提交

### 5.3 资产时间线

**路径**: 资产列表 → 点击任意资产的「时间线」按钮

展示该资产从入库到报废的完整生命周期，包含以下事件类型：

| 事件 | 颜色 | 图标 | 说明 |
|------|------|------|------|
| 入库 | 绿色 | Box | 资产首次入库 |
| 出库 | 蓝色 | Upload | 资产领用/调拨出库 |
| 维修 | 橙色 | Tools | 资产送修 |
| 返修入库 | 绿色 | Download | 维修完成返回入库 |
| 报废 | 红色 | Delete | 资产报废处置 |

每条记录显示时间戳、事件类型、详情描述、操作人。

### 5.4 资产盘点

**路径**: 资产管理 → 资产盘点

**操作步骤**:
1. 选择盘点范围（部门/分类/全部）
2. 系统生成盘点清单
3. 逐项核对，标记结果：
   - **正常**: 账实相符
   - **盘盈**: 实际数量多于账面（自动生成盘盈入库记录）
   - **盘亏**: 实际数量少于账面（自动生成盘亏出库记录）
4. 填写备注说明
5. 点击「提交盘点结果」批量确认

## 6. 库存管理

### 6.1 库存查询

**路径**: 库存管理 → 库存查询

查看所有在库资产，支持按仓库、分类、关键词筛选。

### 6.2 入库管理

**路径**: 库存管理 → 入库管理

**操作步骤**:
1. 选择入库类型：采购入库 / 调拨入库 / 归还入库 / **盘盈入库**
2. 选择目标仓库
3. 选择或输入资产信息
4. 填写数量、备注
5. 点击「确认入库」

### 6.3 出库管理

**路径**: 库存管理 → 出库管理

**操作步骤**:
1. 选择出库类型：领用出库 / 调拨出库 / 维修出库 / 报废出库 / **盘亏出库**
2. 选择资产
3. 填写目标部门/人员、数量、备注
4. 点击「确认出库」

### 6.4 领用审批

**路径**: 库存管理 → 领用审批

这是 3.0 新增的完整审批工作流模块，分为三个标签页：

#### 6.4.1 我的申请（所有用户可见）

- 查看自己提交的所有领用申请
- 点击「新增申请」提交新的领用请求：
  1. 在搜索框输入关键词搜索资产
  2. 选择目标资产
  3. 填写申请原因
  4. 点击「提交」
- 点击「导出」导出申请记录

#### 6.4.2 待审批（审批人可见）

- 显示需要审批的申请列表
- 「通过」：直接批准申请，资产状态变为待出库
- 「拒绝」：弹出拒绝原因输入框，填写后驳回申请

**审批权限**:
- 超级管理员/资产管理员：可审批所有申请
- 部门主管：只能审批本部门的申请

#### 6.4.3 待出库（管理员可见）

- 显示已审批通过但尚未出库的申请
- 点击「确认出库」：
  1. 系统自动创建出库记录
  2. 资产状态从"在库"变为"使用中"
  3. 资产归属更新为申请人部门

#### 6.4.4 审批状态说明

```
提交申请 → 待审批(pending)
       ↓ 通过          ↓ 拒绝
   已通过(approved)   已拒绝(rejected)
       ↓ 确认出库
   已出库(delivered)
```

## 7. 维保管理

### 7.1 维修记录

**路径**: 维保管理 → 维修记录

**新增维修**:
1. 选择故障资产
2. 选择维修类型（必选其一）：
   - **保修期内维修**: 在保修期内的免费维修
   - **保外维修**: 超出保修期的付费维修
   - **厂商送修**: 送回原厂维修
   - **自行维修**: 内部技术人员自行处理
3. 填写维修方式、故障描述
4. 点击「提交」

**更新维修状态**:
- 维修中 → 维修完成：维修人员完成维修
- 维修完成时系统自动通知资产管理员

**返修入库（关键操作）**:

> ⚠️ **重要变更**: 维修完成 ≠ 资产入库。须手动点击「返修入库」。

1. 在维修记录中找到状态为"维修完成"的记录
2. 点击「返修入库」
3. 确认返修日期
4. 资产状态恢复为"在库"

### 7.2 报废管理

**路径**: 维保管理 → 报废管理

**新增报废**:
1. 选择报废资产
2. 选择报废原因：

   | 报废原因 | 必填字段 | 验证规则 |
   |----------|----------|----------|
   | 自然老化 | 确认年限匹配（开关） | 必须开启 aging_match 开关 |
   | 人为损坏 | 责任人 | 必填，记录损坏责任人 |

3. 选择报废日期
4. 填写备注
5. 点击「提交」

提交后，资产状态自动更新为「已报废」。

## 8. 报表统计

### 8.1 折旧报表

**路径**: 报表统计 → 折旧报表

**页面结构**:
- **汇总卡片**: 原值总额 / 累计折旧总额 / 净值总额
- **操作栏**: 「执行折旧计算」按钮 + 「导出」按钮
- **数据表格**: 每项资产的折旧明细

**折旧计算方法**:
- **直线法**: 月折旧额 = 原值 ÷ 使用年限 ÷ 12
- **一次性**: 全额折旧

**操作步骤**:
1. 确保资产已填写「使用年限」和「折旧方法」
2. 进入折旧报表页面
3. 点击「执行折旧计算」
4. 系统批量计算所有在库/使用中资产的折旧数据
5. 查看汇总和明细
6. 需要时点击「导出」下载 CSV 文件

### 8.2 库存统计

**路径**: 报表统计 → 库存统计  
按仓库、分类维度统计库存数量和金额。

### 8.3 出入库报表

**路径**: 报表统计 → 出入库报表  
按时间段统计入库/出库记录，支持筛选操作类型。

### 8.4 资产汇总

**路径**: 报表统计 → 资产汇总  
按部门/分类多维度汇总资产分布。

## 10. 通知中心

### 10.1 通知铃铛

**位置**: 顶部栏右侧，用户头像左侧

**外观**:
- 有未读通知时：铃铛图标显示红色数字角标
- 无未读通知时：仅显示铃铛图标

### 10.2 通知类型

| 类型 | 图标颜色 | 触发场景 | 点击后跳转 |
|------|----------|----------|------------|
| 保修到期 | 橙色 | 资产保修期即将到期 | — |
| 库存预警 | 红色 | 库存低于预警阈值 | 库存查询 |
| 审批通知 | 蓝色 | 新领用申请/审批结果 | 领用审批 |
| 维修完成 | 绿色 | 维修工单完成 | 维修记录 |

### 10.3 使用方式

1. 点击铃铛图标，弹出通知列表
2. 列表显示最近的未读通知
3. 点击某条通知：
   - 自动标记为「已读」
   - 跳转到相关业务页面
4. 系统每 30 秒自动刷新未读计数

## 附录 C: 常见问题

**Q: 为什么我不能提交领用申请？**  
A: 仅「普通用户」和「部门主管」角色可以提交领用申请。管理员请使用普通用户账号测试。

**Q: 维修完成后资产为什么没回到在库状态？**  
A: 3.0 版本将「维修完成」和「返修入库」分离为两步操作。维修完成后需手动点击「返修入库」。

**Q: 为什么我的折旧计算结果是 0？**  
A: 请确认资产登记时已填写「使用年限」字段，且资产状态为「在库」或「使用中」。

**Q: 如何批量导入资产？**  
A: 资产列表页点击「导入」→ 下载模板 → 按模板填写数据 → 上传文件 → 预览确认 → 执行导入。
